Privacy Policy

Last updated: March 13, 2026

Refine Station ("the Extension") is a Chrome extension that helps you refine and improve text using the Google Gemini API. This privacy policy explains what data the Extension handles and how it is used.

1. Data We Collect

We do not collect, store, or transmit any personal data to our own servers. The Extension does not have a backend server operated by us. All processing happens locally in your browser or via direct API calls from your browser to Google.

2. Data Stored Locally

The Extension stores the following data locally in your browser using Chrome's built-in storage (chrome.storage.sync):

  • Your Gemini API key — provided by you and used to authenticate requests to the Google Gemini API.
  • Your preferences — tone, language, detail level, selected model, and enabled/disabled state.

This data is synced across your Chrome browsers if you are signed into Chrome sync. It is never sent to us or any third party other than Google (as part of Chrome sync).

3. Data Sent to Third Parties

When you click "Refine" or "Writing Tips," the text you have typed in the active text field is sent directly from your browser to the Google Gemini API (generativelanguage.googleapis.com) along with your API key. This is necessary to provide the text refinement functionality.

We do not intercept, log, or store this data. Google's use of your data is governed by Google's Gemini API Terms of Service and Google's Privacy Policy.

4. Permissions

The Extension requests the following browser permissions:

  • storage — to save your API key and preferences locally using Chrome's built-in storage.

The Extension also uses a content script that runs on all websites to detect text input fields (textareas and contenteditable elements) and display the "Refine" and "Writing Tips" buttons. The content script only reads text from fields you explicitly choose to refine by clicking a button.

5. Data Security

Your API key is stored using Chrome's built-in storage mechanism. All communication with the Google Gemini API is conducted over HTTPS. We recommend that you do not share your API key with others.

6. Children's Privacy

The Extension is not directed at children under 13 and does not knowingly collect personal information from children.

7. Changes to This Policy

We may update this privacy policy from time to time. Changes will be reflected on this page with an updated "Last updated" date.

8. Contact

If you have questions about this privacy policy, please contact us at redefine.b4it.to.
